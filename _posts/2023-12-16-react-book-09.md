# 9. 컴포넌트 스타일링

- 컴포넌트 스타일링은 다양한 방식을 사용할 수 있고 회사마다 사람마다 다르기에 몇가지 방식을 예시로 듬

  - 일반 CSS : 가장 기본적인 방식
  - Sass : 자주 사용되는 CSS전처리기 중 하나로 확장된 문법사용
  - CSS module : 스타일을 작성할 때 파일마다 고유한 이름을 자동으로 생성해 주는 옵션
  - styled-components : 스타일을 자바스크립트 파일에 내장시키는 방식으로 스타일을 작성함과 동시에 해당스타일이 적용된 컴포넌트 생성

- 이파트는 이런 패턴이 주로 쓰이는구나 정도를 알고 필요할 때 상세히 찾아보는게 효율적일것 같다.
